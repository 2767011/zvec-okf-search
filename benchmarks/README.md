# Контроль качества поиска

Каталог `corpus/` содержит публичный искусственный OKF-корпус, а
`queries.json` — 50 русскоязычных запросов. Поэтому результат можно
воспроизвести без доступа к частной базе знаний.

## Локальный запуск

```bash
okf-zvec index \
  --okf benchmarks/corpus \
  --db benchmarks/db \
  --model e5

okf-zvec benchmark \
  --file benchmarks/queries.json \
  --db benchmarks/db \
  --model e5 \
  --modes semantic,fts_raw,fts,hybrid
```

Запуск против работающего сервиса:

```bash
okf-zvec benchmark \
  --file benchmarks/queries.json \
  --service-url http://127.0.0.1:8765 \
  --token-file ./search-token \
  --model e5 \
  --modes semantic,fts_raw,fts,hybrid
```

## Метрики

- `top1`, `top3` и `mrr` оценивают позицию первого подходящего результата.
- `recall_at_k` учитывает все документы, указанные в `relevant_contains`.
- `ndcg_at_k` учитывает полноту и порядок релевантных результатов.
- `average_ms` измеряет задержку без серверного кэша (`no_cache=1`).

FTS-релевантность нормализуется из BM25 и зависит от состава корпуса. Поэтому
порог `min_relevance` следует калибровать на своём наборе документов; значения
между разными корпусами нельзя считать напрямую сопоставимыми.

## Базовый результат

Дата: 2026-07-15. Zvec 0.5.1, `multilingual-e5-small`, CPU, `topk=5`, порог
релевантности `0.25`. Модель прогрета до начала измерения.

| Режим | Top-1 | Top-3 | MRR | Recall@5 | nDCG@5 | Среднее |
| --- | ---: | ---: | ---: | ---: | ---: | ---: |
| `semantic` | 96% | 98% | 0.967 | 98% | 0.970 | 36.2 мс |
| `fts_raw` | 68% | 76% | 0.717 | 75% | 0.724 | 4.2 мс |
| `fts` | 100% | 100% | 1.000 | 100% | 1.000 | 0.6 мс |
| `hybrid` | 98% | 100% | 0.990 | 100% | 0.993 | 19.8 мс |

Корпус намеренно содержит ясную русскую терминологию, поэтому FTS здесь очень
силён. Эти цифры служат регрессионной точкой, а не доказательством превосходства
одного режима на произвольных данных.

При одинаковом BM25 лемматизация подняла Top-1 с 68% до 100% и устранила 12
пустых выдач. Это отдельный измеряемый вклад русской морфологической нормализации.
